# 🌟 Almdrasa - Bond To-Do Project

تطبيق قائمة مهام ذكي، آمن، وعصري للغاية مصمم باللغة العربية ومبني باستخدام أحدث تقنيات الويب (**TypeScript**, **Vite**, **Sass**). يجمع التطبيق بين **التصميم البصري الفاخر (Glassmorphic UI)** والأداء العالي المعتمد على **Event Delegation**، مع حماية فائقة ضد هجمات **Cross-Site Scripting (XSS)**.

---

## 🎨 المميزات والتصميم البصري (UI/UX)

تم تصميم واجهة المستخدم بعناية فائقة لتقديم تجربة استخدام ممتعة وجذابة بصرياً:
1. **تأثير الزجاج (Glassmorphic Layout):** تصميم بطاقة المهام الرئيسية بخلفية شبه شفافة مع تمويه للخلفية (`backdrop-filter: blur(20px)`) وإضاءات ملونة متوهجة (`.Bubble`) في الخلفية.
2. **شريط تقدم ذكي (Live Progress Indicator):** مؤشر تفاعلي يعرض نسبة إنجاز المهام في الوقت الفعلي مع شريط تعبئة انسيابي ملون متدرج (Gradient fill).
3. **التعديل المباشر للمهام (In-place Task Editing):** تجربة فريدة تتيح للمستخدم تعديل نص أي مهمة مباشرة عن طريق النقر المزدوج (Double Click) عليها أو الضغط على أيقونة التعديل، مع إمكانية الحفظ بـ `Enter` أو الإلغاء بـ `Escape`.
4. **تأثيرات حركية دقيقة (Micro-animations):**
   - دخول انسيابي وسلس للمهام الجديدة عند الإضافة (`@keyframes slideIn`).
   - أيقونة الوضع الداكن/المضيء تدور وتتغير بشكل تفاعلي بالاعتماد على Sun/Moon SVGs.
   - خانات اختيار (Checkboxes) دائرية تفاعلية تعرض علامة الصح بحركة تكبير تدريجية.
5. **أيقونات عصرية مخصصة:** تم دمج أيقونات SVG فائقة الدقة والوضوح لشبكات التواصل الاجتماعي في الهيدر، تشمل الشعار الجديد لمنصة **Twitter/X** وتكامل شبكة **LinkedIn** مع روابط الحسابات الشخصية الخاصة بك.
6. **أيقونة المفضلة (Favicon):** تصميم أيقونة قائمة مهام مخصصة بصيغة SVG متدرجة الألوان وعالية الدقة لتظهر في لسان المتصفح.

## 🔒 التحسينات الأمنية (Security Enhancements)

* **الحماية من هجمات XSS:** تم استبدال عملية الحقن المباشر للمدخلات بفلترة وتشفير كامل لنصوص المهام المدخلة عبر دالة `escapeHTML` مخصصة لمنع أي محاولة لحقن أكواد JavaScript خبيثة عبر قيم المهام (مثل `<img src=x onerror=alert(1)>`).

---

## ⚡ تحسينات الأداء والسرعة (Performance Upgrades)

* **تقنية تفويض الأحداث (Event Delegation):** بدلاً من ربط مئات مستمعي الأحداث بكل عنصر ومهمة على حدة عند كل عملية تحديث (والتي تسبب استهلاكاً كبيراً للذاكرة وبطئاً مع كثرة المهام)، تم ربط **مستمع أحداث واحد فقط** بالحاوية الرئيسية `<ul>` لتوزيع ومعالجة أحداث النقر، الحذف، التعديل، والتفاعل عبر الكيبورد بطريقة فائقة السرعة والأداء.
* **البناء السريع (Vite Bundler):** التطبيق يبني نسخته الإنتاجية في ثانية واحدة بحجم ملفات صغير جداً (CSS ~11KB و JS ~6KB) مما يضمن سرعة تحميل البرق.

## 💻 التشغيل المحلي (Local Development)

لتشغيل المشروع محلياً على جهازك، اتبع الخطوات التالية:

1. **تثبيت الحزم البرمجية:**
   ```bash
   npm install
   ```
2. **تشغيل خادم التطوير المحلي:**
   ```bash
   npm run start
   ```
   *سيعمل خادم التطوير على الرابط المحلي:* `http://localhost:5173/`

3. **بناء النسخة الإنتاجية المجهزة للنشر:**
   ```bash
   npm run build
   ```

## 🚀 النشر على GitHub Pages (Deployment)

تم تجهيز المشروع بالكامل للنشر بنقرة واحدة على GitHub Pages تحت اسم المستودع الجديد **`bond-todo`**:

1. **اربط مشروعك المحلي بالمستودع الجديد على جيت هاب:**
   ```bash
   git add .
   git commit -m "feat: complete professional portfolio upgrade"
   git remote add origin https://github.com/sbondok/bond-todo.git
   git push -u origin main
   ```
2. **انشر تطبيقك مباشرة على الإنترنت:**
   ```bash
   npm run deploy
   ```
   *سيتوفر تطبيقك بشكل مباشر في غضون 30 ثانية على الرابط التالي:*
   👉 **[https://sbondok.github.io/bond-todo/](https://sbondok.github.io/bond-todo/)**
